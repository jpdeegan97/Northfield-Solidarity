# 002 TAXONOMY

## Entities
### Financial Institution
- Bank / lender
- VC / PE / private credit firm
- Fund / vehicle
- Desk/team (optional)
- Affiliate/subsidiary relationships

### Deal
- Lending / credit facility
- Equity raise
- M&A / acquisition financing
- Structured finance / securitization
- Project finance
- Government program loans/grants (if included)

### Counterparties
- Borrower / issuer / target company
- Co-investors / syndicate members
- Underwriters / arrangers
- Advisors (optional)

## Instruments
- Revolver / term loan / mezzanine / bridge
- Bonds / notes / convertibles
- Equity (seed, series, growth) / preferred
- Asset-based lending
- Project finance structures

## Data provenance classes
- Public: filings, press releases, company sites, regulator data
- Licensed: paid deal databases (must track license + usage limits)
- User-provided: operator uploads, internal notes

## Confidence tiers
- High: direct disclosure + matching identifiers
- Medium: multiple corroborations, partial identifiers
- Low: inferred; requires review before publishing
