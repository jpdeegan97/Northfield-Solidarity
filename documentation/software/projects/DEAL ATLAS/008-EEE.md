# 008 EEE (Events, Errors, Escalations)

## Events
- institution.added / institution.verified / institution.archived
- deal.discovered / deal.normalized / deal.published / deal.updated / deal.archived
- backfill.started / backfill.completed
- alert.new_deal / alert.deal_update

## Errors
- connector.blocked_by_policy
- extraction.failed
- entity_resolution.ambiguous
- schema.validation_failed
- licensed_usage_violation (hard stop)

## Escalations
- Licensed usage violation → stop connector + notify operator
- Sudden extraction error spikes → pause and investigate
- High ambiguity rate for a source → require review gate
