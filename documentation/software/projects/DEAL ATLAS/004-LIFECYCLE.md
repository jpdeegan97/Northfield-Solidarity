# 004 LIFECYCLE

## Institution lifecycle
DRAFT → VERIFIED → ACTIVE → MONITORED → ARCHIVED

## Deal lifecycle
DISCOVERED → EXTRACTED → NORMALIZED → RESOLVED → PUBLISHED → UPDATED → ARCHIVED

## Update lifecycle
- new deal detected
- terms updated/amended
- refinancing event
- repayment/default indicators (public only)

## Retention
- Keep evidence snapshots per policy and licensing constraints
- Preserve changelog for published deal records
