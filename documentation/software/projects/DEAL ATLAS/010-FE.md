# 010 FE

## UI modules
1) Institution search + profile
- overview: total deals, total $ deployed (with caveats), trends
- sector/geo/instrument distribution
- top counterparties and repeat relationships

2) Deal explorer
- faceted search (date, amount, instrument, sector, geo)
- deal detail page with citations and changelog

3) Network view
- syndication graph (institution ↔ institution)
- institution ↔ company relationship graph
- filters by time window and deal type

4) Alerts
- subscribe by institution, sector, geography
- daily/weekly digest

## UX rules
- Always show provenance (public/licensed/user-provided)
- Confidence indicator on each deal
- Click-through citations for all key fields
