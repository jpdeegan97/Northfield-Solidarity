# 009 IMPL

## Implementation plan

### Phase 0 — Skeleton (days)
- Institution registry + manual deal CSV upload
- Deal schema + evidence storage
- Basic search and filters

### Phase 1 — Public-source ingestion (weeks)
- Add 3–5 public connectors (filings, press, official announcements)
- Extraction + normalization + entity resolution
- Backfill for initial institution list

### Phase 2 — Analytics + dashboards (weeks)
- Institution dashboard (deployments by sector/geo/instrument/time)
- Deal explorer with facets
- Relationship graph (syndication/co-invest networks)

### Phase 3 — Change detection + alerts (ongoing)
- Monitor sources continuously
- New deal notifications
- Amendments/refinancing detection

### Phase 4 — Licensed sources (optional, only if permitted)
- Add licensed connectors with strict policy enforcement
- Track quotas, attribution, and retention rules
