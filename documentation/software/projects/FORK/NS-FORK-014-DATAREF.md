## Trust & Safety (Chef program baseline)
**Vetting (examples):**
- Identity verification (where required)
- Background check (jurisdiction-dependent)
- Food safety certification validation (e.g., ServSafe or equivalent)
- Proof of insurance (general liability; add professional coverage as needed)
- Reference checks / prior employment verification (optional but recommended)

**Sanitation & Session Controls:**
- Pre-session checklist: handwashing, surfaces sanitized, clean tools
- Allergen handling protocol: separate tools/areas when needed
- Temperature controls: safe cook/hold ranges, labeled storage
- Post-session handoff: leftovers labeling + storage guidance
- Attestation: chef signs completion and any deviations

**Privacy Controls:**
- Address sharing limited to confirmed booking
- In-app messaging with masking
- Minimum necessary data exposure to provider
