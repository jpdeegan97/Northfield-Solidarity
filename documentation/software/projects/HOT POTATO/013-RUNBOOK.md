# 013 RUNBOOK

## Runbooks

### Connector onboarding
1) ToS/robots review
2) Rate limits/backoff defined
3) Allowlist approval
4) Regression samples captured
5) Stage → prod rollout

### Create and run a CWP timeline (operator)
1) Verify ICP fit + review brief evidence
2) Confirm decision-maker path (Personnel Map)
3) Choose strategy template (value-first, signal-triggered, referral wedge, etc.)
4) Select value artifact (mini-audit/benchmark/teardown)
5) Approve steps, timing, and channels
6) Execute touches and log outcomes
7) If “no” or opt-out: stop immediately and suppress

### Objection handling (ethical)
- Ask permission to clarify, provide proof/value, and exit gracefully if not interested.

### Deliverability/bounce spike
- Pause sending, verify email sources, reduce volume, adjust copy, warm domains.

### Opt-out / takedown
- Suppress identity globally; delete Tier 2 data per policy; retain suppression token; audit.
