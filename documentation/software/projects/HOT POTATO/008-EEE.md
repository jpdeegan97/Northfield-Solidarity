- cwp.asset.drafted / cwp.timeline.reviewed / cwp.timeline.activated

## Errors
- connector.blocked_by_policy
- entity_resolution.ambiguous
- evidence.missing_for_claim (hard fail for approval)
- sensitive_data.detected (requires review; Tier 3 blocked by default)
- export.failed
- deliverability.bounce_spike

## Escalations
- opt_out / complaint → immediate suppression + pause related timelines
- high bounce rate → stop sending, validate sources, tighten throttles
- deceptive/manipulative copy attempt → block and require rewrite/review
