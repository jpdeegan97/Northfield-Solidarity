# 009 IMPL

## Implementation plan (with CWP baked in)

### Phase 0 — Skeleton (1–2 days)
- CSV ingest + one allowlisted connector
- Normalize + dedupe + evidence store
- Stub brief + scoring

### Phase 1 — Safer Leads Mode (1–2 weeks)
- Add 3–5 allowlisted connectors
- Evidence-backed brief generator + scoring
- Operator review + CSV export

### Phase 2 — CWP (1–3 weeks)
- Generate Company Report + Personnel Map + Conversion Timeline
- Draft outreach assets (email/call scripts) for operator review
- Track outcomes (reply/meeting/opt-out/bounce)

### Phase 3 — Signals + sequencing improvements (ongoing)
- Better intent signals and freshness weighting
- Strategy effectiveness tracking (which sequences work by segment)
- “Best next action” suggestions

### Phase 4 — Guardrailed automation (later)
- Optional auto-send only for low-risk, high-confidence sequences
- continuous deliverability + complaint monitoring
