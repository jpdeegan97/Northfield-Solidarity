# NS-BOOM-004-LIFECYCLE — Lifecycle

## Run Lifecycle
1. **Ideate**: choose market + hypothesis
2. **Specify**: create RunSpec + success criteria
3. **Build**: minimal assets + instrumentation
4. **Launch**: go live (timebox starts)
5. **Iterate**: bounded tweaks (copy/targeting/offer) with logs
6. **Close**: stop spend/outreach, freeze assets
7. **Report**: postmortem + scorecard + recommendations
8. **Archive**: store artifacts and telemetry
9. **Learn**: update strategy library + next-run suggestions

## Library Lifecycle
- Strategy Template drafted
- Used in runs
- Scored and refined
- Graduated to “recommended”
- Deprecated when consistently underperforming or channel changes
