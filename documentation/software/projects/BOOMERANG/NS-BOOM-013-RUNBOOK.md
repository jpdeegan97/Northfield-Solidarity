# NS-BOOM-013-RUNBOOK — Runbook (How to Operate Boomerang)

## Pre-Run Checklist
- Define market slice narrowly.
- Choose one primary strategy; limit variable changes.
- Define success criteria and decision gates.
- Set spend cap and timebox.
- Prepare minimal assets (landing/copy/scripts).
- Install instrumentation (events, UTM, lead capture).
- Confirm policy gates (brand/outreach/data handling).

## During Run
- Monitor daily (or hourly for short timeboxes).
- Apply bounded iterations:
  - copy tweaks
  - audience refinements
  - offer framing adjustments
- Log every change with rationale.
- Capture qualitative signals continuously.

## Closeout
- Stop spend and outreach.
- Freeze assets and targeting snapshots.
- Export telemetry.
- Produce report within 24 hours.

## Postmortem Template (minimum)
- Hypothesis
- What shipped
- What happened (numbers)
- What surprised us
- Objection clusters
- What we’d do next
- Verdict: double-down / pivot / park / kill
