## FE surfaces
### Client Portal (MVP)
- Login / magic link
- Session list
- Bundle view:
  - download (zip / pdf)
  - view changelog (v1.0 → v1.1)
  - request iteration day
  - request deletion

### Operator Console (optional)
- Today’s sessions
- Capture checklist + metadata entry
- Upload status + retry
- Pipeline job status (ingest/transcribe/synthesize/export)
- Redaction + publish screen (QA)

## UX principles
- “Fast access to bundle” beats fancy dashboards
- Make version history obvious and non-confusing
- Clear privacy controls (download logs + delete request)
