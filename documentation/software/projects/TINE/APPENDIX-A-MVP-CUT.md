# Appendix A — Practical MVP Cut (Fastest Path)

## MVP Goal
Generate a curated menu from user intent and produce **multiple optimized carts** with transparent tradeoffs.

## MVP Scope
- Intent Builder (duration, healthiness slider, diet/allergies, budget/time)
- Menu generation (7–21 day plans; 2–4 meals/day)
- Ingredient ledger (dedupe + unit conversion)
- Store search + offer matching (limited set of partners)
- Cart optimization (Cheapest / Fastest / Fewest Stores / Balanced)
- Substitution confirmation flow
- Basic feedback loop (like/dislike, save favorites)

## City/Chef Feature — MVP2
- Chef profiles + application flow
- Credential verification + sanitation checklist
- Booking + escrow payment
- Ratings + incident workflow
