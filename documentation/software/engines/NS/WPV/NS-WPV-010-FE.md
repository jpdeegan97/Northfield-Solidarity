# 010 FE

## UI Modules
1. **Upload & Job Status**
2. **Paper Reader**
   - PDF viewer with span overlays
   - Clickable anchors from visuals → highlight source

3. **Graph Explorer**
   - Pan/zoom, search, filter
   - Toggle layers: semantic / math / doc structure

4. **Equation Explorer**
   - Render equations
   - Show variable dependencies
   - Jump to where variables are defined

5. **Review Panel**
   - Accept/reject suggestions
   - Merge/split nodes
   - Edit labels + attach citations

6. **Export Center**
   - Choose views
   - Generate bundle
   - Download/share

## UX Defaults
- Always show confidence + citations.
- “Unverified” content visually distinct.
- Keyboard shortcuts: search, next claim, next equation.
