# NS-CON-004-LIFECYCLE — Lifecycle

## Connector Lifecycle
Draft → Implemented → Tested → Published → Active → Deprecated → Retired

## Connection Lifecycle
Created → Validated → Enabled → Running → (Paused | Error) → Disabled → Deleted

## Run Lifecycle
Scheduled/Triggered → Running → (Succeeded | Partial | Failed) → Archived

## Schema Lifecycle
Discovered → Versioned → Drift Detected → Migrated/Updated → Confirmed
