1. Purpose

This SOP governs the ownership, licensing, use, and monetization of intellectual property (IP) across the Northfield Solidarity entity ecosystem, ensuring legal protection, tax defensibility, brand consistency, and operational clarity.

This SOP exists to:

  • Clearly define IP ownership versus IP usage
  • Enable lawful and auditable licensing between affiliated entities
  • Protect core brands, systems, and proprietary assets
  • Support scalable monetization of IP across operating entities

Risks mitigated:

  • IP ownership ambiguity or loss
  • Unlicensed or improper IP usage
  • IRS recharacterization of IP-related revenue
  • Brand dilution or infringement exposure

Outcome ensured:

  • Centralized, defensible IP ownership
  • Controlled and documented IP licensing
  • Audit-ready intercompany IP revenue streams

2. Scope

Entities Covered:

  • Northfield Solidarity (Parent / HoldCo)
  • Designated IP-Holding Entity (if separate)
  • All operating subsidiaries, brands, and storefront entities

Roles Covered:

  • IP Owner / IP Holding Entity
  • Legal / Governance Lead
  • Finance Lead / Controller
  • Brand Manager
  • Authorized Signatories

Activities Included:

  • IP identification and classification
  • IP ownership registration and maintenance
  • Intercompany IP licensing
  • Royalty calculation and billing
  • IP usage monitoring and enforcement

Explicit Exclusions:

  • Capital contributions (Capital Flow SOP)
  • Non-IP operational services (Revenue Flow SOP)
  • Cash execution mechanics (Finance & Cash Flow SOP)

3. Definitions

4. Governing Principles

  • IP ownership must be explicit, documented, and centralized where appropriate
  • Usage of IP without a license is prohibited
  • All intercompany IP licenses must be arm’s-length
  • IP licensing is separate from revenue for goods or services
  • Documentation is mandatory and auditable
  • Substance must match form in IP arrangements

5. Roles & Responsibilities

6. Process Overview

Inputs:

  • IP assets (brands, systems, content)
  • License requests from operating entities
  • Royalty benchmarks or valuation inputs

Key Steps:

  1. IP identification and classification
  2. IP ownership registration
  3. License request and approval
  4. Royalty determination
  5. License execution
  6. Ongoing monitoring and enforcement
  7. Royalty invoicing and reconciliation

Outputs:

  • Executed IP license agreements
  • Royalty invoices and records
  • Updated IP register
  • Audit-ready documentation

7. Detailed Procedures

7.1 IP Identification & Registration

  • Action: Identify IP assets and record in centralized IP Register
  • Owner: IP Owner / Legal Lead
  • Output: Updated IP Register with ownership evidence

7.2 License Request & Review

  • Action: Operating entity submits license request specifying scope and use
  • Owner: Licensee Entity / Legal Lead
  • Output: Approved license request

7.3 Royalty Determination

  • Action: Determine arm’s-length royalty based on benchmarks or valuation
  • Owner: Finance Lead with Legal review
  • Output: Documented royalty methodology

7.4 License Execution

  • Action: Execute written IP License Agreement
  • Owner: Authorized Signatories
  • Output: Signed license agreement

7.5 IP Usage Monitoring

  • Action: Monitor compliance with license terms and brand standards
  • Owner: Brand Manager / IP Owner
  • Output: Compliance reports or corrective actions

7.6 Royalty Billing & Reconciliation

  • Action: Invoice royalties per agreement; reconcile payments
  • Owner: Finance Lead / Controller
  • Output: Royalty invoices, payments, and ledger entries

8. Controls & Compliance

  • Written license required before IP usage
  • Dual approval for intercompany IP licenses
  • Arm’s-length documentation retained
  • Periodic review of royalty rates
  • Annual IP audit and register verification

9. Records & Documentation

10. KPIs & Monitoring

  • Licensed vs unlicensed IP usage incidents
  • Royalty collection accuracy
  • Timeliness of royalty invoicing
  • IP audit findings
  • Brand compliance scores

11. Exceptions & Escalation

  • Unauthorized IP use escalated immediately to Legal and Managing Member
  • Royalty disputes escalated to Finance Lead → Managing Member
  • Emergency license granted temporarily with post-facto documentation

12. Tools & Systems

  • IP Register / Management System
  • Document Management System
  • Accounting & Invoicing Software
  • Brand Monitoring Tools
  • Proprietary IP management system (future)

13. Security & Access Control

  • View access: Legal, Finance, Brand Management
  • Edit access: Legal / IP Owner
  • Approval authority: Authorized Signatories / Managing Member
  • Audit logs required for all license changes

14. Audit & Review

  • Annual IP audit and register review
  • Royalty rate review annually or upon material change
  • Triggered review upon new brand, system, or market entry
  • Responsible Party: Legal / Governance Lead
